# Helios Native Bridge (Terminal)

A native bridge tool to replicate the Helios Portal (https://portal.helioschain.network/bridge) using Linux terminal.

## Features

- Bridge tokens from Helios → Sepolia and Sepolia → Helios
- Ethers.js v6
- CLI-controlled
- Secure with .env

## Setup

```bash
npm install
cp .env.example .env
